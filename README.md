 Designed for You 🎨

موقع بسيط لعرض دورات تعليمية باستخدام HTML و CSS.

## 📌 فكرة المشروع
هذا المشروع عبارة عن موقع يعرض مجموعة من الدورات (Courses) بشكل منظم مع صور لكل دورة.

## 🛠️ التقنيات المستخدمة
- HTML
- CSS

## 📂 محتويات المشروع
- index.html → الصفحة الرئيسية
- style.css → تنسيق الموقع
- course1.html → صفحة الدورة الأولى
- course2.html → صفحة الدورة الثانية
- course3.html → صفحة الدورة الثالثة
- course4.html → صفحة الدورة الرابعة
- الصور الخاصة بالدورات

## 🚀 طريقة التشغيل
1. حمل المشروع أو انسخه
2. افتح ملف index.html في المتصفح

أو شغّله باستخدام Live Server في VS Code
